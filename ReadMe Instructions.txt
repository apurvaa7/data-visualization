To view and use the dashboard please:
 1. Open a command prompt window where you have saved the unzipped folder.
 2. Run the following command:
    If you have Python3 installed, python -m http.server 8000
    If you have Python2 installed, python -m SimpleHTTPServer 8000
 3. Go to localhost:8000/dashboard.html in your browser.

This version uses dummy data. For the text analysis part, 
only data for years 2009 and 2010 has been populated.

If the map does not render properly, please try resizing the window. This is a known 
JavaScript bug.
